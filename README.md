# All-Pro Asphalt & Concrete Services - GitHub Pages Version

This is a static HTML/CSS/JS version of the website, ready for GitHub Pages.

## File structure

- `index.html` - main page
- `styles.css` - all styling
- `script.js` - small reveal animation
- `assets/favicon.svg` - site icon

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload all files and folders from this project.
3. Commit and push to the default branch.
4. In GitHub, open **Settings > Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select your branch and set the folder to `/root`.
7. Save.
8. GitHub will publish the site and provide a live URL.

## Edit business info

Update these places if needed:

- phone number in `index.html`
- email address in `index.html`
- service area text in `index.html`
- colors and spacing in `styles.css`
